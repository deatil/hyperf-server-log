## Server-log


### 项目介绍

*  基于swoole，使用hyperf框架开发的服务日志系统
*  接收日志及分析日志


### 环境要求

 - PHP >= 7.3.0
 - Hyperf >= 2.1.0
 - Swoole >= 4.1.0
